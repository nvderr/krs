# Krs VS Code Extension

Syntax highlighting, autocomplete, snippets and hover docs for `.krs` files.

## Install

```bat
install.bat
```

Reload VS Code: `Ctrl+Shift+P` → **Developer: Reload Window**

## Features

- Colored syntax (keywords, strings, functions, modules)
- Autocomplete for keywords, `@stdlib/...`, `krs:...`
- Snippets (`fn`, `if`, `for each`, `class`, `use`)
- Hover help on keywords
